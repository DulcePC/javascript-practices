Las variables son basicamente almacenamientos, donde guardaremos x tipos de datos que necesitemos

var producto = "Monitor de 24 pulgadas";

De esta manera inicias una variable y le asignas un valor. Tambien puedes inicializar una variable sin valor y asignarlo despues

var disponible;
disponible = true;

Las variables no pueden iniciar con numeros

Algo que hay que tener bastante pendiente es que las variables se pueden reasignar, para esto existen cierto tipos de variables que son:

var name;
let name;
const name = "My name";

Las variables var y let

Pueden ser reasignadas, practicamente son iguales. Las variables Let son nuevas de ES6 y si ambas se parecen pero tienen ciertas diferencias.

LET: al igual que var la variable let puede ser actualizada dentro del mismo bloque donde se utilize. Una variable let no puede ser re-declarada dentro de su ambito

let saludar = "dice Hola";
saludar = "dice hola tambien";

Esto va a devolver un error diciendo que "Saludar" ya a sido declarado.

Pero a esto me refiero que puede ser reasignada dentro de su mismo ambito

let saludar = "dice hola";
if(true) {
let saludar = "dice hola tambien";
console.log(saludar); // "dice hola tambien"
}
console.log(saludar); // "Dice hola"

A diferencia de var aqui no tendremos que preocuparnos de errores ya que ambar instancias son tratadas como diferentes.

Const

Estas variables mantienen sus valores. Se pueden acceder a ellas dentro del bloque en el que fueron declaradas. No pueden actualizar ni volver a declarar. Es basicamente lo contrario de que hemos visto anteriormente.
