Scope

Los arrays son listas de informacion en la cual podemos utilizar ciertos metodos para obtener sus valores, estos son:

  • push
  • pop
  • slice
  • Map
  • Filter
  • forEach
  • fill

Son variables especiales que pueden almacenar mas de un valor y tambien puede contener valores de diferentes tipos.

var frutas = ["manzana","banana", 40];

Veamos algunos ejemplos de los arrays methods:

El metodo pop() elimina el ultimo elemento del array.

El metodo push() Agrega un elemento al array pero al final.

El metodo unshift() Agrega un elemento al array pero al inicio.
