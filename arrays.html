Funciones

Las funciones son ordenes de procesos de se deben ejecutar

Procedimientos para realizar x accion y/o tarea. Las funciones siempre deben retornar algo.

function addTwo(number) {
return number + 2;
}

addTwo(3);

Estos reciben variables como parametros. Los parametros son como variables que reciben cualquier valor que necesites para realizar tu procedimiento, a su ves tenemos los argumentos que son los valores que le enviamos a nuestra funcion una vez ya han sido declaradas.

En el ejemplo planteado arriba, los parametros son lo que estan dentro del parentesis (number) y el argumento viene siendo cuando llamamos la funcion addTwo(3).
